# SONAR Rock vs Mine Prediction with Python

## Overview
This project builds a Logistic Regression machine learning model to classify underwater objects as either Rocks or Mines based on SONAR signal data.

## Problem Statement
Submarines use SONAR signals to detect underwater objects. The objective of this project is to predict whether an object is a Rock or a Mine using machine learning.

## Dataset
- 208 observations
- 60 numerical sonar features
- Binary classification
  - R = Rock
  - M = Mine

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## Machine Learning Workflow
- Data Loading
- Data Exploration
- Feature & Target Separation
- Train/Test Split
- Logistic Regression
- Model Evaluation
- Prediction

## Results
- Training Accuracy: **83.42%**
- Testing Accuracy: **76.19%**

- The model predicted the sample input as: **Mine**
